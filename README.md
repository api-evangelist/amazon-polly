# Amazon Polly

Amazon Polly is a cloud service that converts text into lifelike speech, enabling you to create applications that talk and build entirely new categories of speech-enabled products.

## APIs

- **Amazon Polly API** - The Amazon Polly API enables you to synthesize speech from text, manage lexicons for custom pronunciations, and list available voices across multiple languages.
